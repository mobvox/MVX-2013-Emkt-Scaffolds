# EMKT-Scaffolds

## Usage

```emktscaffold MVX-2013-Nome-Email --langs en es pt-BR```

## Output

```
/MVX-2013-Nome-Email
/MVX-2013-Nome-Email/img
/MVX-2013-Nome-Email/{lang}/index.html
/MVX-2013-Nome-Email/{lang}/img
```
## Supported languages
- es
- pt-BR
- en

## Installation

```python setup.py install```
